Simple boilerplate for Koa & TypeScript.
I referred to [javieraviles/node-typescript-koa-rest][repo].

[repo]: https://github.com/javieraviles/node-typescript-koa-rest

# Requirements

- yarn or npm
- node 14+

## Installation

```bash
$ yarn
```

## LINT, Test

```bash
$ yarn lint
$ yarn test
```

## Start the server

### dev

```bash
$ yarn dev
```

### prod

```bash
$ yarn build
$ yarn start
```
